关于四维遐想

发表于 2017-03-02 | | 阅读次数

杨熊子的朴素哲学观。以下纯属胡言乱语胡思乱想。

最近在学高数，上来第一章说的是空间几何，第二章是多元函数。众所周知，维度和变元的概念是十分相似的。在数学上的几何知识是从一维推广到二维三维以及多维的，并且几个维度就能用几个变元构成的方程表示。

这里引用高数老师酷酷的一句话：

从一维到二维是质变，从二维到多维只是量变。

这里我想，从多维到无限维又是质变。（这句话与本文无关，只是作者为了装酷）

那么作为一个一直在观察宇宙的中二少年，维度的扩展就很令我瞎遐想了。

以前一直在想四维空间，四维世界是怎么样的？

《三体》和《星际穿越》都让我对四维空间着迷，以致于画坐标系的时候，我总是忍不住从原点画出一条螺旋线，不断地贯穿三个坐标轴，像螺旋星云一样往外延伸。

四维空间会不是像电影胶带一样，按照时间一帧一帧的连续，每一帧都是当时世界的模样。我们在不断往随历史往前演进。

那样历史、命运是不是注定的？人——三维生物有可能返回到以前的帧数中吗（穿越时空）？时间是不是连续的？无限可分的？时间有没有一个终点？还有那个坐标原点…是不是可以通过它窥探四维空间？……

PS：写这篇nonsense的时候，我眼前全部是马修在四维空间处看着女儿的情景。

哇，囊空如洗！——忍太郎《大富翁4》
